# 📦 Smart Logistics Company Landing Page

Современный и функциональный лендинг для международной логистической компании, базирующейся в Бишкеке. Проект ориентирован на высокую конверсию, удобство использования на мобильных устройствах и эстетичный визуальный стиль.

## 🚀 Как запустить
Проект не требует установки дополнительных зависимостей или сборки:
1. Скачайте файл `smart-logistics.html`.
2. Откройте его в любом современном браузере.

## ✨ Особенности
* **Modern UI/UX:** Использование премиальных шрифтов *DM Serif Display* и *DM Sans*.
* **Responsive Design:** Полная адаптивность под смартфоны и планшеты.
* **Interactive Elements:**
    * Анимации появления элементов при скролле.
    * Плавная навигация по разделам.
    * Кастомное мобильное меню.
    * Интерактивная форма заявки с обратной связью.
    * Бегущая строка (Marquee) с ключевыми услугами.

## 🛠 Технологии
* **HTML5** — семантическая разметка.
* **CSS3** — Grid, Flexbox, Custom Properties (переменные), Animations.
* **JavaScript (Vanilla)** — логика интерфейса без внешних библиотек.
